{{"Connect to an instance"|i18n}}

{{"Prerequisites"|i18n}}

  1. {{"Enable SSH port"|i18n}} -- {{"Open the instance's SSH port (usually on port 22)"|i18n}}.
  2. {{"Instance's public SSH DNS"|i18n}} -- Domain, port.
  3. {{"Private key"|i18n}} -- {{"Get the path to your private key in your local machine."|i18n}}

  1. {{"Your key file must not be publicly viewable for SSH to work.Use this command if necessary"|i18n}}

    chmod 400 {{keypair}}.pem

  2. {{"Connect to your instance using its Public SSH DNS."|i18n}}

    ssh -i {{keypair}}.pem root@{{domain}} -p {{port}}

  1. {{"Get PuTTY and PuTTYgen."|i18n}}

    {{"Official"|i18n}} http://www.chiark.greenend.org.uk/~sgtatham/putty/download.html

  2. {{"Convert your private key file."|i18n}}

    PuTTYgen can convert {{keypair}}.pem to .ppk format.

  3. {{"Connect using PuTTY SSH."|i18n}}

    {{"Enter Host name"|i18n}} {{domain}} {{"and port"|i18n}} {{port}}, {{"import"|i18n}} {{keypair}}.pem {{"file"|i18n}}.
